### 背景

新冠病毒全球蔓延，身在欧洲，异国它乡的华人，需要有一个平台，能够相互间
- 发布救助信息
- 发布自己可以提供的资源信息
- 报平安

基于此，`Together`(华人互助平台)项目诞生了

### 项目介绍

`ichina` 是 `Together` 的后台接口项目

#### 基本技术栈
- `ruby on rails` 
- `mysql`

#### 结构及部署

- `app/controllers/api` 里面放的是接口的代码
- `mina producion deploy` 就可以部署到远程的生产服务器

#### 文档访问
- 项目集成了swagger在线文档，地址： `localhost:3000/api/doc`
